# Smart To-Do List Android App

Smart To-Do List is a simple Android application built using **Kotlin and Room Database** that allows users to manage daily tasks efficiently.

The app provides a clean and minimal interface where users can add tasks, mark them as completed, and track when the tasks were created.

---

## Features

- Add new tasks
- Mark tasks as completed using a checkbox
- Completed tasks show a strike-through effect
- Timestamp showing when the task was added
- Task list displayed using RecyclerView
- Local storage using Room Database

---

## Tech Stack

- Kotlin
- Android Studio
- RecyclerView
- Room Database
- Material Design Components

---

## How the App Works

1. The user enters a task in the input field.
2. Pressing the **Add Task** button saves the task in the Room database.
3. The task appears in the list with the timestamp.
4. The user can mark the task as completed using the checkbox.
5. Completed tasks show a **strike-through text effect**.
